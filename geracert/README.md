# Geracert 2.0

## Este projeto consiste em um gerenciador de certificados para participantes em eventos de radioamadorismo. Ele armazena em banco de dados as informações da participação e pode gerar o certificado em formato PDF baseando-se nestes dados armazenados, informações de layout e uma imagem de fundo predefinida pela organização do evento. 
## Enfim, a idéia é evitar o problema que tinhamos antes de precisar gerar manualmente um arquivo para cada participante ou mesmo gerar com dados variáveis em Corel Draw etc.
## Com o Geracert, o objetivo principal é diminuir o consumo de espaço no armazenamento online e manter a emissão de certificados sempre disponível. 
## A versão inicial estava escrita em PHP sem uso de framework, porem comecei a encontrar problemas na organização e reaproveitamento de código, dessa forma o Geracert foi reescrito com framework Codgniter 4.

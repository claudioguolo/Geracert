# Changelog

All notable changes to this project will be documented in this file.

## v1.0.0

Initial public release prepared for the `Geracert` GitHub repository.

Highlights:

- public search by callsign and club
- PDF certificate generation
- admin dashboard with roles
- contest template visual editor
- CSV certificate import
- image upload with validation rules
- multilingual UI support for `pt-BR` and `en`
- database migrations and seeders
- automated feature tests
- documentation for deployment and template customization
